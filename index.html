Выражения и операции в PHP

Выражения

Выражение - все что имеет какое-то значение в PHP.

Логические выражения

Логические выражения - это выражения, у которых могут быть только два значения, ложь или истина. 

                    $less = 10 < 5; // $less - false
                    $equals = $b == 1; // $equals - true, если $b ==1
                    $between = $b >= 1 && $b <=10 // $between - true, если $b от 0 до 10
                    $x = ! ($b || $c) && $d; //  true, если $b и $c ложны, а $d - истинно

Строковые выражения

Строки один из основных объектов PHP

Строка в апострофах

Строка в апострофах трактуется так как записана, исключение сам апостров и обратный слеш. Для них требуется экронирование обратным слешом.

Строка в кавычках

Для строк в кавычках предусмотренны сециальные метасиволы.

\n - новая строка \r - возврат каретки \t - табуляция $ - переменная \" - кавычка \nXX - символ с шестнадцатеричным кодом XX

Объединение строк

$SOME = "Hell"; // слово Hello без буквы "o" echo $SOME."o world!"; // один способ echo "{$SOME}o world!"; // другой способ echo "${SOME}o world!"; // третий способ!
